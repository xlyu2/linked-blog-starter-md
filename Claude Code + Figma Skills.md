## Figma Skills: 
- Figma Skills Page: https://www.figma.com/community/skills
- wildly inconsistence, not really speed up my workflow(yet)
- figma skill are structured files that guide AI on how to work with Figma
- Define how to do specific actions: creating components, generating layouts, or syncing variables
- reusable instructions for AI

## Connect Figma MCP to Claude
- claude code -- customize -- connectors -- figma
- popup browse window hit agree
- drag in skills - figma-use
- open a claude chat, add all inside the reference folder, figma-use skill
	- prompt:"add these to the figma-use skill, inside of a references folder"
	- "add all of these to a working-with design-system-folder inside of references"
